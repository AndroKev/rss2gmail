rss2imap
========

An adaptation of rss2email that uses GMAIL directly (with the help of feedparser)

Based on [rcarmo](https://github.com/rcarmo/rss2imap)



# Main Differents to rss2imap:
- works only with gmail
- support subfolder(labels)
